# Chihuahua-Muffin-Hackathon-bvvcodres
# Chihuahua vs Muffin Classification using Deep Learning and 3LC

## Overview
This project was developed for a hackathon to classify images as either a **Chihuahua 🐶** or a **Muffin 🧁** using Deep Learning techniques.  

The project uses:
- Python
- PyTorch
- 3LC Dashboard
- Ensemble Models
- Data Curation and Analysis

The system was trained on a curated dataset and evaluated using accuracy metrics and embedding visualizations through 3LC.

---

# Features
- Binary image classification
- Deep learning model training
- Ensemble prediction system
- Automated prediction pipeline
- 3LC dashboard integration
- Embedding visualization
- Accuracy analysis
